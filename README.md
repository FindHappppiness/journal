# 행찾 백엔드

## 💿 Tech Stack
Python 3.9
Django 4.2

## 🚀 Get Started

1. Repository Clone
```
   git clone https://github.com/FindHappppiness/journal.git
   cd YOUR_PATH/journal
``` 
2. Python 가상환경 생성
```
   py -3.9 -m venv venv
   cd YOUR_PATH/journal/venv/Scripts
   activate
```
3. Requirements 설치
```
   cd YOUR_PATH/journal
   pip install -r requirements.txt
```

4. 실행
```
   python manage.py runserver
```
